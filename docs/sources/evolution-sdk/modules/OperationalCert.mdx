## OperationalCert (class)

OperationalCert class based on Conway CDDL specification

CDDL:

```
operational_cert = [
  hot_vkey : kes_vkey,
  sequence_number : uint64,
  kes_period : uint64,
  sigma : ed25519_signature
]
```
